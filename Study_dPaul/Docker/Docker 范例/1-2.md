# 安装 MySQL

**Docker官网地址** [MySQL](https://hub.docker.com/_/mysql)

## 基础镜像

**拉取镜像**
```bash
$ docker pull mysql:5.7.31
```

**备份镜像**
```bash
$ docker save mysql:5.7.31 -o mysql.5.7.31.tar
```

**导入镜像**
```bash
$ docker load -i mysql.5.7.31.tar
```
