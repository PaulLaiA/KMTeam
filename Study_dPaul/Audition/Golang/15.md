# Ans
都是格式化輸出

但是目的不一樣

| Func        | 目標                               |
| ----------- | ---------------------------------- |
| `Printf()`  | 一般是屏幕，也可以重定向           |
| `Sprintf()` | 把格式化字符串输出到指定的字符串中 |
| `Fprintf()` | 把格式化字符串输出到文件中         |
