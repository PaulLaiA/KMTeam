# Node

在**Metadata.Type**中**Node**代表的是一個節點

一般用於定義象徵文件夾的文件

Ex :

`Study_dPaul/Appendix/Metadata` 文件夾下有一個與文件夾同名的文件 `Metadata`

那這個文件其實就代表這個文件夾

這種設計模式保留了既有的文件夾分類的方法,同時可以讓每個文件夾内的資料使用一種節點的方式進行互通
