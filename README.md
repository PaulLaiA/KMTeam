# 使用教學

## 工具介紹

1.  Obsidian
	主力寫筆記的工具
	
	下載網址:https://obsidian.md/
	
	中文基礎教學:https://publish.obsidian.md/help-zh
	
	Obsidian 使用外掛
	- 核心外掛
		- 模板
	- 第三方外掛
		-  Admonition
		- DataView
		- Editor Syntax Highlight
		- Outliner
		- QuickAdd

2. VSCode
	彌補 Obsidian 對於檔案沒辦法實現一些批量級操作
	Notepad++ 也是可以,這個隨意
3. SourceTree
	使用 Git 的小工具
4. 簡繁轉換的小工具
	https://github.com/lxn/walk
