
# 笔记大纲

```dataview
table 分类,创建日期,file.etags as 标签
from "面試區/面試題"
where !contains(file.name,"大纲")
sort file.ctime
```
---

# 今日新增
```dataview  
table 分类,创建日期,file.etags as 标签
from "面試區/面試題"
WHERE 创建日期 > date(today) - dur(1 day) 
and !contains(file.name,"大纲")
sort file.ctime
```

# 今日修正
```dataview  
table 分类,创建日期,file.etags as 标签
from "面試區/面試題"
WHERE file.mday > date(today) - dur(1 day) 
and !contains(file.name,"大纲")
sort file.mday
```
