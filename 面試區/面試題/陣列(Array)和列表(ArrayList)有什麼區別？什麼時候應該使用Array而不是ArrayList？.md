## 區別

### 不同點

| 區別         | Array              | ArrayList                                                     |
| ------------ | ------------------ | ------------------------------------------------------------- |
| 包含型別     | 基本型別和對像型別 | 對像型別                                                      |
| 大小         | 固定               | 動態變化                                                      |
| 方法         | 基礎方法           | 提供更多特性及方法                                            |
| 基本數據類型 |                    | 使用自動裝箱來減少代碼量,固定大小的數據類型的時候就會比較緩慢 |
| 多維數組     | 比較方便           | 比List嵌套方便                                                |
